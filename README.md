## 5. 易宿酒店预订平台 - 管理端
	
## 项目介绍
本项目是易宿酒店预订平台的管理后台，用于商户管理酒店信息和管理员审核发布。

## 技术栈
- React 18
- Ant Design 5
- React Router v6
- json-server (模拟后端)
## 功能模块
- 登录/注册：支持商户和管理员角色切换。
- 商户功能：录入、编辑酒店信息。
- 管理员功能：审核、驳回（带理由）、发布、下线酒店。
- 数据可视化：首页展示审核状态饼图。
- 操作历史：记录酒店审核的全流程轨迹。
## 启动方式
1. 启动后端：`cd server && npx json-server --watch db.json --port 3001`
2. 启动前端：`npm run dev`
