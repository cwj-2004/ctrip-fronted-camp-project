# 易宿酒店预订平台 - 第五期前端训练营大作业

## 1. 项目简介

本项目为一个综合性酒店预订系统，包含：

- **用户端 (Mobile)**: 移动端预定流程，包含查询、列表及详情页。
- **管理端 (PC)**: 酒店信息录入、审核及发布系统。
